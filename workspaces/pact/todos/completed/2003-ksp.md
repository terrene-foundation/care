# TODO-2003: KnowledgeSharePolicy (KSP)

Status: pending
Priority: high
Dependencies: [1001, 2001]
Milestone: M2

## What

Implement `KnowledgeSharePolicy` per PACT-REQ-005. A KSP is the explicit grant that allows knowledge to flow across an organizational boundary. The thesis states clearly: **absence of a KSP IS the barrier** — there is no default sharing, no implicit trust between D/T nodes. The burden is on the creator to grant access, not on the requester to prove they need it.

KSP semantics per thesis Section 6.3:

- A KSP is created by the **lowest common ancestor** of the source and target units in the D/T/R hierarchy. If Trading (D1-R1-T2-R1) wants to share with Advisory (D1-R1-T3-R1), the creator must be at D1-R1 or above. A role that does not contain both parties cannot create a KSP for them.
- `source_unit`: D or T address of the knowledge-owning unit.
- `target_unit`: D or T address of the unit being granted access.
- `max_classification`: the maximum `ConfidentialityLevel` this policy covers. A KSP for PUBLIC data does not grant access to CONFIDENTIAL data even between the same two units.
- `created_by`: address of the role that created the policy (must be a common ancestor of both units).
- `expires_at`: optional `datetime`. Expired KSPs are treated as non-existent by the access algorithm — no grace period.
- `policy_id`: unique identifier (same character restrictions as `item_id`).
- `bilateral`: if True, the policy grants access in both directions (source can see target's data up to `max_classification` and vice versa).

The `find_ksp()` helper searches a registry of KSPs for a valid (non-expired) policy between two units that covers a given classification. Returns `None` if no matching policy exists. The access algorithm treats `None` as "access denied" with no fallback.

Validation rules:

- `source_unit` and `target_unit` must be different addresses.
- `created_by` must be a common ancestor of both units (validated via address prefix-containment from TODO-1001).
- `max_classification` must be a valid `ConfidentialityLevel` — no default.
- If `expires_at` is in the past at construction time, raise `KSPExpiredError` immediately.

## Evidence

- KSP between D1-R1-T2-R1 and D1-R1-T3-R1, created by D1-R1, constructs successfully.
- KSP created by D1-R1-T2-R1 (not a common ancestor of T3) raises `KSPAncestorError`.
- `find_ksp(registry, source="D1-R1-T2-R1", target="D1-R1-T3-R1", classification=CONFIDENTIAL)` returns the policy when a valid one exists.
- `find_ksp(...)` returns `None` when no policy exists — confirming absence is the barrier.
- `find_ksp(...)` returns `None` for an expired policy (expired KSP is invisible).
- Bilateral KSP: `find_ksp(source=T3, target=T2, ...)` also returns the policy when `bilateral=True`.
- `max_classification=SECRET` KSP does not match a `TOP_SECRET` classification query.

## Details

```python
from __future__ import annotations
from dataclasses import dataclass, field
from datetime import UTC, datetime
from typing import Any
from pact.build.config.schema import ConfidentialityLevel

_CLASSIFICATION_ORDER: dict[ConfidentialityLevel, int] = {
    ConfidentialityLevel.PUBLIC: 0,
    ConfidentialityLevel.RESTRICTED: 1,
    ConfidentialityLevel.CONFIDENTIAL: 2,
    ConfidentialityLevel.SECRET: 3,
    ConfidentialityLevel.TOP_SECRET: 4,
}

@dataclass(frozen=True)
class KnowledgeSharePolicy:
    policy_id: str
    source_unit: str
    target_unit: str
    max_classification: ConfidentialityLevel
    created_by: str           # Must be common ancestor of source_unit and target_unit
    bilateral: bool = False
    expires_at: datetime | None = None

    def __post_init__(self) -> None:
        if self.source_unit == self.target_unit:
            raise ValueError("source_unit and target_unit must differ")
        if self.expires_at and self.expires_at < datetime.now(UTC):
            raise KSPExpiredError(f"KSP {self.policy_id!r} already expired")
        # validate created_by is common ancestor — uses TODO-1001 address utilities

    def is_expired(self) -> bool:
        if self.expires_at is None:
            return False
        return datetime.now(UTC) > self.expires_at

    def covers(self, classification: ConfidentialityLevel) -> bool:
        """Returns True if this KSP covers the given classification level."""
        return _CLASSIFICATION_ORDER[classification] <= _CLASSIFICATION_ORDER[self.max_classification]

    def to_dict(self) -> dict[str, Any]: ...
    @classmethod
    def from_dict(cls, data: dict[str, Any]) -> KnowledgeSharePolicy: ...


def find_ksp(
    registry: list[KnowledgeSharePolicy],
    source: str,
    target: str,
    classification: ConfidentialityLevel,
) -> KnowledgeSharePolicy | None:
    """Find a valid KSP granting source access to target's knowledge at the given level.

    Returns None when no matching non-expired policy exists.
    Absence of a return value IS the access barrier — callers must deny on None.
    """
    for ksp in registry:
        if ksp.is_expired():
            continue
        if not ksp.covers(classification):
            continue
        # Direct match
        if ksp.source_unit == source and ksp.target_unit == target:
            return ksp
        # Bilateral match
        if ksp.bilateral and ksp.source_unit == target and ksp.target_unit == source:
            return ksp
    return None
```
